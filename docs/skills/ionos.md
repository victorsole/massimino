Se connecter à IONOS (Login)
La connexion à IONOS se fait via différentes interfaces selon le produit. Cet article vous aide à trouver le bon login et à résoudre les problèmes de connexion.

Remarque

En cas de problèmes de connexion, veuillez d'abord vérifier si vous utilisez les données d'accès correctes pour le produit concerné. Faites ici la distinction entre le mot de passe de votre compte IONOS et les produits individuels protégés par un mot de passe, comme les boîtes e-mail.

Compte IONOS
Dans le compte IONOS, vous gérez vos contrats et les produits qui y sont liés, comme les domaines, les solutions d'hébergement Web, les adresses e-mail et les serveurs.

Login :  https://login.ionos.fr
Connexion : La connexion se fait avec votre Numéro de client, l'adresse e-mail de contact enregistrée ou avec l'un de vos domaines.
Vous avez oublié votre mot de passe ? Si vous avez oublié le mot de passe de votre compte IONOS, vous pouvez en créer un nouveau sur la page suivante :
https://motdepasse.ionos.fr
Vérifier et modifier l'adresse e-mail de contact :

Vous trouverez l'adresse e-mail enregistrée dans votre compte compte IONOS sous Menu > Mon compte > Mes données client > Contact. Vous pouvez la modifier à tout moment.

Sécurité : Code de confirmation par e-mail

Si vous n'avez pas activé l'authentification à deux facteurs (2FA), nous vous envoyons dans certains cas, pour des raisons de sécurité, un code à 6 chiffres à votre adresse e-mail de contact lors de la connexion. Saisissez ce code dans le champ correspondant pendant le processus de connexion afin de confirmer l'accès à votre compte.

Le code est valable pendant 20 minutes.
Si le code n'est plus valable, vous devez faire une nouvelle tentative de connexion.
Pour votre protection, les codes sont transmis et traités de manière cryptée.
Webmail
Webmail est une application basée sur un navigateur et spécialement conçue pour l'utilisation des boîtes aux lettres électroniques IONOS.

Login :  https://mail.ionos.fr
Connexion : La connexion est possible avec votre adresse e-mail IONOS complète et le mot de passe correspondant.
Vous avez oublié votre mot de passe ? Si vous avez oublié le mot de passe d'une boîte e-mail IONOS, vous ne pouvez le réinitialiser que via votre compte IONOS. La procédure à suivre est décrite dans l'article suivant du Centre d'Assistance : Modifier le mot de passe d'un compte e-mail chez IONOS
HiDrive
HiDrive est un espace de stockage en ligne de IONOS, qui vous permet de partager des données avec d'autres utilisateurs et appareils.

Login :  https://hidrive.ionos.com
Vous avez oublié votre mot de passe ? Si vous avez oublié le mot de passe de HiDrive, vous pouvez en demander un nouveau. L'article suivant du Centre d'Assistance décrit la procédure à suivre : Demander un nouveau mot de passe
Data Center Designer (IONOS Cloud)
Le Data Center Designer (DCD) vous permet de gérer votre compte IONOS Cloud. Dans le DCD, vous créez de nouveaux centres de données virtuels et éditez vos projets.

Login :  https://dcd.ionos.com/
Si la connexion échoue
Si la connexion échoue, vérifiez d'abord ces sources d'erreur courantes :

Erreurs de frappe : Faites attention aux caractères spéciaux et aux majuscules/minuscules.
Touche de verrouillage : Vérifiez que la touche de verrouillage (Caps Lock) est activée.
Pavé numérique : Assurez-vous que le pavé numérique est actif. Vous pouvez l'activer en appuyant sur la touche de verrouillage numérique.
Cache du navigateur : Utilisez une fenêtre privée (mode incognito) à titre d'essai ou videz le cache du navigateur.
Perturbation temporaire : Sur la page de statut IONOS, vous pouvez vérifier s'il y a une perturbation.
Si la connexion n'est toujours pas possible, prenez contact avec le Service Client IONOS.
